## Breaking Changes

```typescript
// Old authentication method (deprecated)
const client = new Client({
  apiKey: "your-api-key"
})

// New OAuth 2.1 method
const client = new Client({
  clientId: "your-client-id",
  clientSecret: "your-client-secret",
  redirectUri: "https://your-app.com/callback"
})
```
